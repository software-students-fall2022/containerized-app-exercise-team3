# Web App

## To Run the Web App (Without Container)

### Create `.env`

In `web-app/`, create the `.env` file.

### Install All Dependencies

In `web-app/` run:

    pip3 install -r requirements.txt

### Run `flask`

In `web-app/` run:

    flask run

## To Run Unit Tests

### Run `pytest`

In `web-app/` run:
    
    python3 -m pytest
